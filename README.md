# Design Patterns
Design patterns are common ways of solving recurring problems. In this repository, I'll be implementing them using Kotlin aiming to improve my studies and my knowledge.

## Implementations
### Creational
- Factory Method
- Abstract Factory
- Builder
- Prototype
### Structural
- Adapter
- Bridge
- Decorator
- Composite
- Flyweight
- Proxy
### Behavioral
- Chain of Responsibility
- Command
